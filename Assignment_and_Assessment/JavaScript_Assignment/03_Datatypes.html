Q.3 Define a Data Types in js?

JavaScript is a dynamically typed language, meaning that variable types are determined at runtime. It supports several primitive data types and objects:

  1. Primitive Data Types:
    • String: Represents textual data.
    • Number: Represents numeric values, including integers and floating-point numbers.
    • Boolean: Represents true or false values
    • Undefined: Represents an uninitialized or undefined value
    • Null: Represents the intentional absence of any object value
    • Symbol: Introduced in ES6, represents a unique and immutable identifier.
  2. Objects:
    • Object: A complex data type that can hold key-value pairs and methods.
  3. Special Data Types:
    • Function: A callable object that can be defined and invoked. JavaScript's flexibility allows developers to work with a combination of these data types, making it suitable for a wide range of programming tasks, from simple scripting to complex web application development.
