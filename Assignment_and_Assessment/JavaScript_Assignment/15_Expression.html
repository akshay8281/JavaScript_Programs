Q.15 What is the result of the expression (5 > 3 && 2 < 4)?

Ans -

The expression '(5 > 3 && 2 < 4)' is a logical expression in JavaScript that uses the logical AND operator ("&&"). The expression evaluates to "true" because both conditions on either side of the "&&" operator are true:

"5 > 3" is true.
"2 < 4" is true.

When both conditions connected by the logical AND operator are true, the overall result is "true". Therefore, the result of the expression "(5 > 3 && 2 < 4)" is "true" .
