Q.30 What are the looping structures in JavaScript? Any one Example?

Ans -

JavaScript provides several looping structures to repeatedly execute code. One fundamental structure is the "for loop," particularly useful when the number of iterations is known in advance. Its syntax involves three components: initialization, condition, and update. For example:

for (var i = 0; i < 5; i++) { console.log(i); }

Here, the loop initializes a variable i to 0, continues as long as i is less than 5, and increments i after each iteration. The output will be:

Result
0 1 2 3 4

Another commonly used structure is the "while loop," suitable when the exact number of iterations isn't predetermined.It repeats as long as a specified condition remains true:

var i = 0; while (i < 5) { console.log(i); i++; }

This loop achieves the same result as the "for loop." Additionally, the "do-while loop" guarantees the execution of the block at least once before evaluating the condition. These looping structures are essential for handling repetition in JavaScript, enabling efficient and flexible code execution.
