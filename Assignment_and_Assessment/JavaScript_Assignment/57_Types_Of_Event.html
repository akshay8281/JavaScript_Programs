Q.57 how many type of JS Event? How to use it?

Ans -

JavaScript events are actions or occurrences that happen in the browser, and they can be handled by JavaScript code. There are various types of events in JavaScript, broadly categorized into:

Mouse Events:

  1. Mouse Events:
    • click: Occurs when a mouse button is clicked.
    • mouseover/mouseout: Triggered when the mouse pointer enters/exits an element.
    • mousedown/mouseup: Fired when a mouse button is pressed/released.
  2. Keyboard Events:
    • keydown/keyup: Fired when a key is pressed/released.
  3. Form Events:
    • submit: Triggered when a form is submitted.
    • input: Fired when the value of an input element changes.
  4. Document and Window Events:
    • load: Fired when a page has finished loading.
    • unload: Occurs when the user navigates away from a page.
    • resize: Triggered when the browser window is resized.
    • scroll: Fired when an element's scrollbar is scrolled.
  5. Focus Events:
    • focus/blur: Fired when an element gains/loses focus.
  6. Event Delegation:
    • Utilizes event propagation to handle events on parent elements rather than individual child elements
