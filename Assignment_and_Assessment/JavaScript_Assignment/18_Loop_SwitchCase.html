Q.18 What is a Loop and Switch Case in JavaScript define that ?

Ans -

A loop is a control flow structure in programming that allows a set of instructions to be repeatedly executed until a certain condition is met. JavaScript provides several types of loops, including `for`, `while`, and `do-while`.

`for` loop:** Executes a block of code a specified number of times. It has three parts: initialization, condition, and increment/decrement.

for (let i = 0; i < 5; i++)
{ // Code to be repeated }

`while` loop:** Repeats a block of code as long as a specified condition is true.

let i=0;
while (i < 5) {
// Code to be repeated i++; }

`do-while` loop: Similar to `while` loop, but it guarantees that the code block is executed at least once before checking the condition.

let i=0;
do {
// Code to be repeated i++; } while (i < 5);

Switch Case in JavaScript:

A switch statement is used to perform different actions based on different conditions. It evaluates an expression against multiple possible case values and executes the code associated with the first matching case.

let day=3;
switch (day) {
case 1: console.log("Monday");
break;
case 2: console.log("Tuesday");
break;
// ... other cases default: console.log("Invalid day"); }

In this example, if `day` is `3`, it will execute the code block associated with `case 3`. The `break` statement is crucial to prevent fall-through, meaning execution would continue into the next case without it. The `default` case is optional and is executed when none of the cases match.
