16. How can a page be forced to load another page in JavaScript?

Ans -

In JavaScript, you can use the window.location object to force a page to load another page by changing the URL. Here are two common ways to achieve this:

  1. Setting window.location.href:

    2. This method changes the entire URL, triggering a page load.

    window.location.href = 'https://www.example.com/newpage.html';

  2. Using window.location.replace():

    3. This method replaces the current page with a new one in the browser's history

    window.location.replace('https://www.example.com/newpage.html');

Choose between them based on whether you want the user to be able to navigate back to the original page or not. The replace() method is often preferred when you want to prevent the user from going back to the previous page.
