12. How can you convert the string of any base to an integer in JavaScript?

Ans -

In JavaScript, use parseInt(string, radix) to convert a string from any base to an integer. The string is the input string, and radix is an integer between 2 and 36, representing the base (e.g., 2 for binary, 8 for octal, 16 for hexadecimal). For instance, to convert a binary string "101010" to an integer, use parseInt("101010", 2), resulting in 42. Similarly, adjust the radix for octal or hexadecimal strings. If no radix is specified, the default is 10. For non-integer strings or floating-point numbers, consider using parseFloat instead of parseInt.
