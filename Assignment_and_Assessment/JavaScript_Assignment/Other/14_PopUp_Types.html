14. What are all the types of Pop up boxes available in JavaScript?

Ans -

In JavaScript, there are three main types of pop-up boxes that you can use for user interactions:

  1. alert() Box:
    • Displays a simple message to the user with an OK button.

    • Syntax

      alert('This is an alert box!');
  2. confirm() Box:
    • Prompts the user with a message and provides OK and Cancel buttons.
    • Returns true if the user clicks OK and false if the user clicks Cancel.

    • Syntax

      const result = confirm('Do you want to proceed?');
  3. prompt() Box:
    • Prompts the user to enter some input. It also provides OK and Cancel buttons.
    • Returns the entered text if the user clicks OK and null if the user clicks Cancel.

    • Syntax

      const userInput = prompt('Please enter your name:', 'John Doe');

These pop-up boxes are simple ways to interact with users, but they can be disruptive, so use them judiciously in user interfaces. Modern web development often uses more sophisticated modal dialogs or custom UI components for better user experiences.
